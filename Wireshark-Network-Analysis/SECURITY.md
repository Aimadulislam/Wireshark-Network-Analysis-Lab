# Security Policy

## Supported Versions

We actively maintain and provide security updates for the following versions of the Wireshark Network Analysis toolkit:

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

We take the security of our network forensic utilities seriously. If you find any vulnerabilities, security bypasses, or potential command injection vectors in our bash, python, or docker configurations, please follow these steps:

1. **Do not** open a public GitHub issue.
2. Email your findings directly to the project maintainers or security leads (e.g., `aimadulislam788@gmail.com`).
3. Include detailed steps to reproduce the issue (PoC), your environment details, and recommended remediation if known.

We acknowledge receipt of all valid submissions within **48 hours** and aim to coordinate a security patch within **14 days**.
